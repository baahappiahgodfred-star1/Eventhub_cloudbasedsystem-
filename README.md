# EventHub

## Overview

EventHub is a cloud-based web platform that centralizes information about technology, educational, business, and cultural events across Algeria.

The application allows users to:

* Register and log in securely
* Browse available events
* Create new events
* Manage their own events
* Access event information from a single platform

## Technologies Used

### Frontend

* React.js
* Axios
* React Router

### Backend

* Node.js
* Express.js
* JWT Authentication
* bcryptjs

### Database

* MongoDB Atlas

### Deployment

* Render (Backend)
* Netlify (Frontend)

## Features

* User Authentication
* Event Creation
* Event Listing
* My Events Dashboard
* Protected Routes
* Cloud Deployment

## Architecture

React Frontend → Express API → MongoDB Atlas
